Данный программный продукт предназначен для расчета основных ключевых параметров газотурбинных установок (Далее - ГТУ) 
простого цикла Брайтона и цикла Брайтона с регенерацией тепла уходящих газов. 
Возможности:
1) Вариативный расчет параметров ГТУ при различных степенях повышения давления в компрессоре и
степени эффективности регенерации тепла уходящих газов за турбиной;
2) Построение графиков зависимостей двух и трех величин;
3) Сохранение всех элементов программы на каждом этапе работы в формате .word / .pdf;
4) Создание автоматических отчетов по ГОСТ 2.105-2019 на каждом этапе работы.

*Данное программное обеспечение является моим первым крупным проектом в сфере программирования и требует очивидных доработок в плане архитектуры 
и оптимизации программного кода. Дата окончания работы над проектом 05.2025.
